前言

本教程内容基于 Django Girls ( https://tutorial.djangogirls.org/zh/ ) 教程而来，使用 Django==2.2 对原 Django Girls 教程内容进行了部分更新。

本教程采用知识共享署名-相同方式共享 4.0 国际许可协议进行许可。

你有没有觉得世界越来越多的是关于技术，而你已经在某种程度上被甩在了后面？ 你有没有想去做一个网站，但是却从未有过足够的动力去开始吗? 你有没有想过软件世界太过复杂，即使你想做一些属于自己的东西？

好的，我们跟你讲一个好消息！编程没有看上去那么复杂，我们将给你展示它是多么的有趣。

本教程不会神奇地将你转变为程序员。 如果你想变得擅长于此，你需要数月乃至数年的时间去学习和练习。 但是我们想展示给你编程和创建网站没有看起来那么复杂。 我们试着用深入浅出的方法解释，所以你不会觉得被技术吓到。

我们希望我们可以让你变得更爱技术，像我们一样！

在本教程里你会学到什么？

需要具备的基础

学习 Django 之前，需要你掌握一定的 基础 python 语法，如果你还没有学过 python，可以学习这个简单的 Python 入门教程。

本教程的 Django 版本为：2.2.6，可以使用的 python 版本为 python 3.5, python 3.6, python 3.7，如果使用的不是以上三个版本的 python，可能会导致一些难以预料的错误。

使用本教程时，需要使用 python 3.5, python 3.6, python 3.7 中的一个。

可以去 https://docs.djangoproject.com/zh-hans/2.2/faq/install/#what-python-version-can-i-use-with-django 查看与 Python 相匹配的 Django 版本。
